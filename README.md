# BurgerSchön Gizli Müşteri Denetim Sistemi

BurgerSchön restoranları için geliştirilmiş kapsamlı bir şube denetim ve kalite kontrol sistemi.

## 🍔 Proje Hakkında

Bu sistem, BurgerSchön şubelerinin gıda güvenliği, hijyen ve operasyonel standartlarını denetlemek ve takip etmek için geliştirilmiş web tabanlı bir uygulamadır.

## 🚀 Özellikler

- Detaylı denetim formları
- Gerçek zamanlı puan hesaplama  
- PDF rapor oluşturma
- Şube bazlı performans analizi
- Majör/minör uygunsuzluk takibi
- Mobil uyumlu tasarım
- PWA desteği

## 🛠️ Kullanılan Teknolojiler

- React.js
- Firebase
- Tailwind CSS 
- PDF-Lib
- Chart.js
- React Router
- React Beautiful DnD

## 📱 PWA Desteği

Uygulama PWA desteği ile mobil cihazlara yüklenebilir ve çevrimdışı çalışabilir.
